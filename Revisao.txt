Questão 1: Existem quatro princípios básicos, quais são? Comente sobre cada um deles, explicando com exemplos.

 

Disponibilidade:é quando um site de comércio eletrônico está disponível para os usuários 24 horas por dia, 7 dias por semana, sem interrupções ou quedas. Isso significa que os usuários podem acessar o site a qualquer momento para fazer compras ou obter informações sobre produtos sem nenhum problema.

 

Integridade:é quando os dados armazenados em um sistema são precisos, confiáveis e mantidos intactos, sem nenhuma modificação ou corrupção. Isso significa que as informações armazenadas são protegidas contra alterações não autorizadas ou erros acidentais, garantindo que a qualidade dos dados seja mantida e que as informações sejam utilizáveis para tomada de decisões e outras tarefas críticas.

 

Confidencialidade:é quando as informações são mantidas privadas e acessíveis apenas para pessoas autorizadas. Isso significa que as informações são protegidas contra acesso não autorizado ou divulgação para terceiros não autorizados, garantindo que as informações permaneçam confidenciais.

 

Autenticidade:é a autenticação de usuário em um sistema de computador ou aplicativo. Por exemplo, quando um usuário insere suas credenciais de login (nome de usuário e senha) em um sistema, o sistema verifica se essas informações são válidas e correspondem a um usuário autorizado antes de conceder acesso ao sistema.

Questão 6:Cite um exemplo de cumprimento de cada princípio de Segurança da Informação. Disponibilidade, Integridade, Confidencialidade, Autenticidade.

 

Disponibilidade: Garantir a disponibilidade dos dados significa que eles devem estar acessíveis sempre que necessário. Um exemplo disso é o backup regular dos dados em um seguro local, de forma a garantir que, em caso de falha do sistema ou perda de dados, as informações possam ser recuperadas rapidamente.

Integridade: A integridade dos dados significa que eles não foram alterados ou alterados de forma não autorizada. Um exemplo de garantir a integridade dos dados é o uso de assinaturas digitais para verificar a segurança de documentos e mensagens eletrônicas.

Confidencialidade: Garantir a confidencialidade dos dados significa que eles são autorizados em sigilo e só podem ser acessados ​​por pessoas autorizadas. Um exemplo disso é o uso de senhas fortes e criptografia para proteger dados sensíveis.

Autenticidade: A segurança dos dados significa que sua origem e autoria são conhecidos e verificáveis. Um exemplo de garantia de segurança dos dados é o uso de certificados digitais para garantir a identidade do remetente em transações online.

Operacional

O que é e para que serve o Kernel de um

Sistema Operacional?

 

Um sistema operacional é um software que gerencia os recursos de hardware e software de um computador ou dispositivo eletrônico. Ele fornece uma interface entre os usuários e o hardware, permitindo que os usuários executem programas e realizem tarefas em seus dispositivos.

 

Os três objetivos principais de um sistema operacional são:

Gerenciamento de recursos: O sistema operacional gerencia os recursos de hardware, como cobertura, memória, dispositivos de armazenamento e entrada/saída, para garantir que cada processo e programa tenha acesso aos recursos necessários para executar suas tarefas.

Facilitar a execução de programas: O sistema operacional fornece uma interface que permite que os usuários executem programas e realizem tarefas em seus dispositivos. Ele também gerencia os programas prolongados entre o hardware, garantindo que os programas possam ser executados de maneira eficiente.

Garanta a segurança: O sistema operacional protege os recursos e dados do usuário, garantindo que apenas usuários autorizados possam acessá-los. Ele também controla o acesso aos recursos e arquivos do sistema, garantindo que programas maliciosos ou usuários mal-intencionados não possam comprometer a segurança do sistema.

Periféricos: são dispositivos externos ao computador que se conectam a ele para realizar funções específicas, como mouse, teclado, impressora, monitor, entre outros. Já os componentes são partes internas do computador, como processador, memória RAM, placa-mãe, disco rígido, entre outros.

Cinco funcionalidades de um Sistema Operacional são:

Gerenciamento de recursos: O sistema operacional gerencia o uso de recursos do computador, como memória RAM, processador e espaço em disco, para garantir que cada aplicativo tenha o que precisa para funcionar corretamente.

Interface com o usuário: O sistema operacional fornece uma interface que permite aos usuários interagir com o computador, como a área de trabalho, menu Iniciar e gerenciador de arquivos.

Gerenciamento de arquivos: O sistema operacional gerencia o armazenamento e recuperação de arquivos no computador, permitindo que os usuários criem, salvem, movam e excluam arquivos.

Gerenciamento de redes: O sistema operacional permite que o computador se conecte a redes, como a internet, e gerencie a comunicação entre dispositivos na rede.

Segurança: O sistema operacional implementa medidas de segurança para proteger o computador e os dados dos usuários, como senhas de usuário
